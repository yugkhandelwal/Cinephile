### 🎯 What You Need

Before starting, get these ready:

1. **TMDB API Key**
   - Get it: https://www.themoviedb.org/settings/api
   - Free account required
   - Takes 2 minutes

2. **Supabase Credentials**
   - Get it: https://supabase.com/dashboard
   - Free account
   - Need: Project URL + anon key

3. **GitHub Account**
   - Sign up: https://github.com/join
   - Free account

4. **Vercel Account**
   - Sign up: https://vercel.com/signup
   - Free account (can use GitHub to sign in)
